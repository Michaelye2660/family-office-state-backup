独立外部评审 EXT-06：量化预筛规则的设计盲点

第一问：你对提问者有何已知信息？

结论：我对提问者的个人或机构背景没有任何可靠信息。

我仅知道：提问者希望对一套长期投资体系的机械预筛规则进行独立外部评审；要求评审设计盲点而非评价个股；要求中文、结论先行，并分别讨论漏筛、规则一致性、阈值假设、簇限额及规则修改。

我不知道提问者是否为规则设计者、投资决策者、研究人员或第三方；也不知道其资金规模、风险偏好、持有期限、换手约束、历史业绩及实际候选名单。因此，以下意见只针对所披露规则的结构，不针对提问者本人或任何证券。

局限申报

未见筛选名单及落选名单，无法判断各类盲点在实际样本中的发生频率，也不能计算误杀率或误入率。

未见系统其余部分，包括ROIC、WACC、有形资本、簇分类的具体定义，数据时间窗口、行业映射、异常值处理、人工复核权限及后续深查流程。

"化妆品公司因营业亏损被排除"并非规则文本中的独立明示条件。它可能是第3条计算的自然结果，也可能意味着存在未披露的隐含盈利门槛。

本评审基于单次会话中的静态规则文本，没有进行历史回测、阈值敏感性分析或候选池对照实验。

因此，以下属于设计风险判断，不是对该体系历史有效性的否定，也不构成投资建议。

总体结论

这套规则的主要优点是简洁、可解释、强调资本回报和数据纪律，适合提高候选池的平均质量；其主要风险则是：

把若干不可跨行业直接比较的会计指标当成统一硬门槛；

通过"有形ROIC"同时承担质量判定和毛利率豁免，造成指标重复使用；

使用静态绝对阈值处理会随利率、通胀、估值和产业结构变化的变量；

将组合层的容量约束提前作用于研究入口，可能形成路径依赖和研究盲区。

它更像一套偏重精确率的规则，而不是兼顾召回率的规则。若体系明确接受大量漏筛，这种取舍可以成立；若目标是尽量发现长期复利型生意，则需要建立持续监测漏筛的机制。

问题一：这套阈值组合系统性会漏掉哪几类优质生意？

结论

系统性漏筛最可能集中在六类：低毛利高周转但非零售的生意、指标口径特殊的资产负债表型生意、处于主动投资期或周期低谷的生意、小市值早期复利者、会计分类导致毛利失真的生意，以及资本密集但高度稳定的生意。

论据

1. 低毛利、高周转，但不被归类为零售的生意

40%毛利率地板默认"优质经济性通常体现为高毛利"。但部分生意通过库存周转、应付账款、规模采购、网络密度或极低资本占用创造高资本回报，毛利率本身并不高。

目前只明示豁免零售业态，会系统性遗漏具有相似经济结构、但行业标签不是零售的分销、撮合、物流、外包运营及成本转嫁型模式。

2. ROIC和毛利率不具备普通工业口径的资产负债表型生意

部分行业的核心经济变量是利差、承保质量、资本充足率、损失率或资产负债匹配，而不是传统意义上的营业毛利和工业企业ROIC。对这些行业套用统一毛利率与投入资本公式，可能得到形式上精确、经济上不可比的结果。

3. 暂时亏损但单位经济性仍然优良的主动投资型生意

营业亏损会使当期ROIC失效或为负，但亏损可能来自新市场扩张、渠道建设、研发投入、客户获取或一次性重组。规则会把"结构性劣化"和"主动加大高回报投资"压缩为同一种结果。

已知化妆品边界案例提示了这一问题，但在未见多期数据及亏损成因的情况下，不能判断该排除是正确过滤还是周期性误杀。

4. 处于行业周期低谷的优秀经营者

周期性行业在低谷期可能同时出现毛利率下降、ROIC转负和自由现金流背离。虽然快速预警旗不直接降权，但质量闸和毛利率地板仍可能直接将其排除。

这会造成一种顺周期偏差：景气顶部时企业更容易通过，估值和盈利均处低位时反而更难进入研究池。

5. 市值低于100亿美元的早期复利者

100亿美元闸门能够提高数据质量和流动性，但会机械排除尚未被市场充分定价、仍有较长增长跑道的中型公司。

由于市值是"股价×股数"，该闸门不是经营质量门槛，而部分取决于市场情绪。优质公司在估值低迷时可能掉出，普通公司在估值泡沫期可能进入。

6. 会计列报方式使毛利率或有形资本失真的生意

收入按总额还是净额确认、某些费用计入营业成本还是销售费用、租赁与外包方式、研发费用化、品牌投入费用化，都会显著改变毛利率和投入资本，却未必改变底层经济性。

同一种经济活动仅因合同形式或会计分类不同，可能位于40%地板两侧。

7. 资本密集、回报不突出但确定性极高的生意

若"优质"包括现金流可预测、需求刚性、资产寿命长和下行风险较低，那么部分受监管或长期合同型生意可能长期只获得略高于资本成本的回报，难以达到15%有形ROIC，却仍可能在适当价格下具有长期投资价值。

这是否属于误杀，取决于体系将"优质"定义为高资本回报，还是定义为高确定性风险调整回报。

反方最强论点

预筛的目的不是覆盖所有好公司，而是用有限研究资源提高命中率。主动放弃低毛利、亏损期、资本密集和小市值公司，可以显著减少价值陷阱、周期误判和低质量候选。

这一论点成立，但有一个必要前提：体系必须明确其愿意承担多高的漏筛率，并为被永久排除的类别保留独立观察通道。否则，"后续还有人工深查"并不能弥补预筛漏失，因为未进入名单的企业根本不会得到深查。

可执行建议

建立一份排除原因账本，按行业和排除条件统计每期落选数量；另设固定规模的"影子样本"，持续跟踪各硬门槛附近的落选者。重点监测：

毛利率在30%至40%但资本周转较高者；

ROIC短期为负但收入留存、单位经济性或增量回报改善者；

市值略低于100亿美元者；

周期低谷中多期平均回报仍较高者。

这样才能判断漏筛是预期取舍，还是规则未被发现的结构性偏差。

问题二：第3条"或"逻辑与第4条豁免机制，是否引入可被无意利用的松动或不一致？

结论

是。主要问题不是存在"或"逻辑本身，而是有形ROIC既能独立通过质量闸，又能再次用于推翻毛利率地板，导致同一个指标被重复计票。

这会削弱两个门槛原本应有的相互独立性，并产生行业分类和资本口径上的无意套利。

论据

1. "或"逻辑允许总资本回报不合格者通过

企业即使在包含商誉及其他资本后未能赚取资本成本，只要有形ROIC超过15%，仍可通过质量闸。

这可能是合理校正，也可能掩盖以下问题：

大量收购产生的商誉被排除；

资产减值后分母缩小；

设备已充分折旧但仍在使用；

大量外包使资产离开资产负债表；

负营运资本或客户预付款使有形投入资本异常低；

企业依靠持续收购，而不是存量业务内生创造回报。

在这些情况下，高有形ROIC不一定等于高经济回报。

2. 有形ROIC在零售豁免中被重复使用

一家低毛利企业可以先凭有形ROIC通过第3条，再凭同一有形ROIC绕过第4条。名义上它通过了"质量＋毛利"两个关卡，实质上可能只通过了一个指标。

因此，对豁免对象而言，毛利率地板不再是独立检验，而成为可被第3条同一证据覆盖的重复关卡。

3. "零售业态"存在边界和分类裁量

线上线下一体化、会员制、平台撮合、批发零售混合、品牌直营和供应链服务等模式，很难只用一个零售标签划分。

如果例外资格在看到结果后才判断，就可能形成无意中的分类迁移：希望保留的公司被解释为零售，其他具有相同经济机制的公司则因行业标签不同而被排除。

4. 豁免只覆盖行业名称，没有覆盖经济机制

真正值得豁免的原因应当是"低毛利、高周转、低资本占用或稳定负营运资本"，而不是企业被归类为零售。

按行业名称豁免，会遗漏相同机制的其他行业，也可能让并不具备高周转优势的零售企业获得宽松待遇。

反方最强论点

有形ROIC正是用来纠正商誉、品牌资产和低毛利高周转模式的会计失真。若强制要求ROIC−WACC和40%毛利率同时满足，会把大量优秀资产轻型生意排除。

这一反方论点成立，所以不宜简单把"或"改为"且"。问题在于，有形ROIC需要成为一个有条件的替代路径，而不是无附加验证的万能通行证。

可执行建议

将规则改成两条预先定义的路径：

**路径A：**经统一经济资本口径调整后的ROIC持续高于WACC；

**路径B：**有形ROIC超过阈值，同时满足多期现金转化良好、非依赖持续收购、资本基数未被减值或负营运资本严重扭曲。

毛利率豁免也应按经济机制定义，例如资本周转率、库存损耗、营运资本结构和多期现金回报，而非按"零售"行业名称定义。

所有豁免应在看具体结果前写明资格条件，并单独记录通过率及后续表现。

问题三：40%毛利、15%有形ROIC、100亿美元市值分别隐含什么假设？何时会失效？

结论

三个数字分别隐含了"高毛利代表优质""15%是跨周期足够高的资本回报""100亿美元代表成熟、可靠且仍有机会"的假设。这些假设均不是恒定事实。

论据

一、40%毛利率

隐含假设：

毛利率可以跨行业比较；

定价权和差异化主要体现在毛利率；

营业成本的会计分类稳定；

原材料、运输及其他转嫁成本占比有限；

高周转不足以完全弥补低毛利。

容易失效的环境：

原材料及物流成本快速通胀，而调价存在时间差；

大量成本按合同直接转嫁给客户；

总额法和净额法收入确认发生变化；

企业改变直营、加盟、外包或垂直整合模式；

分销、零售、物流等以资本周转而非单位毛利创造回报的行业。

40%更适合作为特定行业中的相对质量指标，不适合作为全市场统一的硬地板。

二、15%有形ROIC

隐含假设：

有形投入资本基本覆盖企业真实所需的经济资本；

被排除的商誉、研发、品牌及客户获取投入不需要计入回报分母；

15%在各种利率和风险环境下都代表足够高的超额回报；

资本存量没有因历史成本、充分折旧或资产减值而严重低估；

高回报可以维持，而不是依靠暂时压缩库存、资本开支或维护投入。

容易失效的环境：

利率和风险溢价显著上升，15%相对于资本成本不再特别高；

高通胀使历史成本资产低估，机械抬高ROIC；

大规模并购、商誉减值或资产剥离；

品牌、研发、客户获取等经济投资大量费用化；

负营运资本、外包或租赁导致有形资本分母过小；

企业以推迟维护和资本开支换取短期高回报。

因此，15%不能完全替代ROIC相对WACC的利差，也不应脱离资本口径质量单独使用。

三、100亿美元市值

隐含假设：

达到该规模的公司数据更完整、治理更成熟、流动性更好；

规模能够提高生意存续概率；

100亿美元以上仍有足够长期增长空间；

名义美元阈值在不同年份具有相近经济含义；

市场定价不会严重扭曲候选资格。

容易失效的环境：

整体市场估值大幅扩张或收缩；

高通胀使固定名义阈值逐年变松；

行业估值分化使相同经营规模对应完全不同市值；

优质公司因短期争议或周期低谷跌破门槛；

尚处早期扩张阶段的高质量公司普遍低于门槛。

固定市值线本质上同时筛选经营规模和市场估值，不能被解释为纯粹的质量控制。

反方最强论点

固定阈值的价值正在于稳定、透明和避免过度拟合。若所有阈值都按行业、周期和宏观条件动态调整，规则可能逐渐演化为不可复核的主观系统。

这一论点成立。解决方式不应是无限细分，而是把硬断点改成少数、预先规定的区间或相对口径。

可执行建议

将40%毛利率改为行业内分位数或软警示，不再作为全市场统一否决项；

将15%有形ROIC与WACC挂钩，例如要求超过WACC一定安全边际，或同时满足行业内较高分位；

将100亿美元改为定期重置的市场分位数、实际收入规模或流动性门槛，避免固定名义金额随通胀和估值漂移；

对所有门槛保留"距阈值距离"，避免39.9%与40.0%被视为性质完全不同。

问题四：第7条簇限额设计的失效模式是什么？

结论

第7条最大的失效模式是：**用公司数量代替真实风险敞口，并将组合约束提前变成研究约束。**这可能造成表面分散、实际高度相关，同时强化既有持仓的路径依赖。

论据

1. 数量不等于风险

同一簇两家公司可能都是小仓位，也可能已占组合极大比例；三家不同簇的公司也可能共同暴露于利率、消费、云计算资本开支、监管或同一供应链风险。

"几家公司"不能替代仓位权重、盈利来源和共同风险因子的测量。

2. 簇定义可能错误或滞后

企业会扩展业务、改变收入结构或从产品公司转向平台、服务和金融模式。静态簇标签可能不能反映真正利润来源。

同一家公司也可能横跨多个簇。强行放入单一类别，会掩盖跨簇风险。

3. 形成现有持仓优先的路径依赖

一旦某簇已有两家公司，后来的更优候选只能进入"替换观察"。这相当于让进入时间影响研究资源分配。

若现有持仓受到熟悉度偏差或沉没成本影响，新候选可能长期得不到与老持仓同等深度的分析。

4. 将"是否值得研究"和"是否应当持有"混为一谈

即使组合暂时没有容量，新候选仍可能非常值得深入研究，例如用于建立行业认知、更新估值基准或准备未来替换。

在预筛阶段限制研究对象，会使组合约束反过来压缩认知范围。

5. 可能错过行业整体错价

某个行业在系统性冲击后可能同时出现多家高质量低估候选。固定数量上限会在机会最集中时阻止进一步研究。

反过来，空簇最多纳入三家也不能保证这三家彼此不相关或确实具有代表性。

反方最强论点

簇限额能够降低组合重复下注、控制研究注意力、避免因熟悉某行业而持续堆叠同质公司，也有助于减少组合在单一商业模式上的脆弱性。

这一原则是合理的，但应当主要应用在组合建仓和风险预算层，而不是研究入口层。

可执行建议

将"研究池"和"可持仓池"分开管理：

研究池不因现有持仓数量而拒绝优质候选；

组合层按仓位、收入来源、利润敏感度和共同风险因子设置限额；

每个簇同时保留现有持仓与若干"挑战者"，使用同一评分标准定期比较；

定期重新识别簇，不只依赖行业标签；

对跨簇共同风险另设监控，例如利率敏感、周期敏感、监管敏感和单一供应链敏感。

问题五：若只许加一条、删一条、改一条，如何调整？代价是什么？

一、加一条：增加"影子池与漏筛审计"

建议规则

对每个硬门槛附近的落选者，以及因行业口径不适用而落选的公司，按固定比例建立影子池；影子池不进入正式候选，但按相同周期跟踪其经营结果和后续深查结论，用于评估漏筛率及阈值漂移。

理由

目前所有保护机制都集中在防止误入，缺少发现误杀的反馈回路。影子池不需要立即放宽标准，却能回答最关键的问题：被排除的公司后来是否持续表现出更好的经济质量。

它也能区分"有意放弃的机会"和"规则没有意识到的盲区"。

代价

增加数据维护和人工复核工作；

容易产生事后偏见，只关注后来成功的落选者；

若样本设计不固定，可能被选择性用于为放宽规则辩护。

因此，影子样本的抽样比例、观察期限和评价指标必须事前确定。

二、删一条：删除40%毛利率的全市场硬地板

建议处理

删除"毛利率低于40%即排除"的统一否决规则。毛利率继续保留，但改作行业内比较指标和风险标签，不再独立决定是否进入候选池。

理由

该规则是目前最明显的跨行业结构性偏差来源，也直接导致必须设计零售豁免。删除硬地板后，可以消除大量分类争议，并避免相同经济机制因行业名称不同受到不同待遇。

质量仍可通过资本回报、现金转化、增量回报和竞争稳定性控制。

代价

候选池会扩大；

商品化、缺乏定价权和成本压力较大的公司会增加；

后续研究负担上升；

若其他质量指标定义不严，有形ROIC分母偏小的问题会更加突出。

因此，删除毛利硬门槛必须与第3条的收紧同时发生。

三、改一条：把第3条改为"受约束的双路径"，而不是无条件"或"

建议规则

质量闸满足以下任一路径：

经周期和经济资本调整后的ROIC持续高于WACC并具备明确安全边际；

有形ROIC超过15%，但仅在同时满足多期现金转化良好、非依赖持续收购、资本基数未被重大减值或异常负营运资本扭曲时有效。

理由

该修改保留了有形ROIC纠正会计失真的价值，但取消其作为万能通行证的地位，也避免企业仅因分母极小而获得异常高回报读数。

"持续"应当使用预先规定的多期窗口，而不是在个案出现后临时判断。

代价

数据要求和模型复杂度上升；

对年轻企业、重大转型企业和历史数据不足者不利；

可能误伤依靠高质量并购创造价值的经营模式；

WACC估计本身存在显著误差，设置安全边际也不能消除模型风险。

最终评审意见

这套规则不宜被评价为"过严"或"过松"，更准确的判断是：它在某些行业非常严格，在另一些资本口径特殊的行业可能意外宽松。

最高优先级的三个问题是：

统一毛利率门槛造成的跨行业不可比；

有形ROIC在质量闸和豁免中被重复使用；

缺乏用于测量永久漏筛的反馈机制。

若只实施上述"加一条、删一条、改一条"，我建议按以下顺序落地：

先增加影子池审计，再收紧第3条的替代路径，最后取消40%全市场硬地板。

原因是，在缺少实际候选名单和回测结果时，应先建立能够观察误杀与误入的反馈回路，再根据证据调整阈值。否则，无论放宽还是收紧，都可能只是将一种未经测量的偏差替换成另一种偏差。
