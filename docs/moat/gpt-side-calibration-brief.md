## 第一部分：新增最高优先级研究目标——护城河牢固等级 MDG

**这是自即日起你全部产出中最重要的一项，任何报告缺它即不合格。**

你此前的报告给了护城河类型/是否失守，但缺一个委托人最需要的东西：**这护城河到底多牢、危机深跌时他能多有信心扛住回撤并大胆抄底。** 这不是"是什么护城河"，是"这护城河多难被攻破"。逻辑：**越牢固=越难失守=危机中的暴跌是价格波动而非价值损坏=越敢在深跌处大胆抄底、越能忍受深回撤和长时间水下不割。** 这是"危机时敢不敢抄底"这个决策的地基。

**每份报告必先给护城河牢固等级 MDG（1-5）+ 四要素关键理由（充分明确，不得只给等级）：**
- **MDG-5 磐石**：可预见期内几乎不可能失守（不可复制的结构垄断/监管牌照/网络或基础设施），危机=纯价格波动。要求穿越2008∧2020∧2022三役且特许经营本体无恙的实证。→最敢深跌抄底、最能扛深回撤长水下。
- **MDG-4 深沟**：护城河很强但有单一可识别长期威胁（范式迁移/监管尾部），须盯死那一个信号。
- **MDG-3 有沟**：护城河真实但在被侵蚀或多威胁并存；危机可持有不可重仓抄底，先确认侵蚀未加速。
- **MDG-2 浅沟**：护城河脆弱/依赖易变因素（品牌心智/单一大客户/可被新范式替代）；危机不因便宜抄底，先判失守。
- **MDG-1 沙堡/存疑**：护城河名不副实或坍塌中；便宜=陷阱信号，不抄底。

**四要素关键理由**（每级必给）：①牢固来源（具体什么在保护它，尽量可量化）；②最可能的失守路径（怎么被攻破，写成可观测信号）；③抗回撤历史锚（有无穿越历史危机的实证；无则明说"无历史锚、信心打折"）；④当前侵蚀读数（护城河指纹里最相关那条的方向）。

MDG 与"错杀/失守/中性"三色判定并列产出，不互相替代：三色答"当前市价错没错"，MDG答"护城河牢不牢"。

## 第二部分：接入校准回测制度

自即日起，你的每一次真钱盲跑判定都进入**校准回测**——不是记你说了什么，是记你**说对了没有**。规则如下，与对侧（Claude侧）对称。

### 一、每单必须事前登记（冻结即commit）
每次交卷时，在你的 `predictions.md` 台账登记一行，**在结果已知前锁定**：
1. **护城河牢固等级 MDG（1-5）+ 降级触发信号**（什么发生=当初判高了，机械可判）——**这是登记的首字段，最高优先级**；
2. **冻结概率分布**：三态概率（错杀 / 失守 / 无明显错价），相加=100%；
3. **双向证伪信号**（机械可判，写成"X日前Y≥Z"型）：转错杀的条件组 + 转失守的条件组；
4. **结算日** + 近端观测点（如次季财报日）。
登记的 commit 时间戳就是"没有事后调参"的证据。**事后改 MDG 或冻结概率=作弊，该案作废。**

### 二、结算与记分
- 到结算日或证伪信号触发时，据**公开数据**机械判定实际归类（错杀/失守/中性），不得回头改冻结概率。
- 主指标=**多类 Brier score**（三态 one-hot，越低越好）；辅=**错杀概率桶校准**（你说X%错杀的那批案子，实际几成真是错杀——测过度自信）。
- **"无明显错价/证据不足"防滥用**：结算时若仍无足够证据定色，该判定算对；若结算显示当时可得证据已足以定错杀或失守、而你当时躲进"证据不足"，算错。中性不是免罪符。
- **MDG 校准（审计最重要内容）**：MDG 映射"可预见期内失守概率"（MDG-5→极低…MDG-1→高），事后护城河失守=1/未失守=0，算 Brier。**判高了（高 MDG 却事后失守=overconfident-moat）特别单列、危害权重最高、不与普通案混计**——因为它会诱导危机中重仓抄底一个正在失守的公司，是价值陷阱最致命的形态。**危机终极考试**：当市场跌超20%或个股跌超30%，强制回看你当时在册的高 MDG 标的，护城河是否真扛住（份额/定价权/客户留存三读数）——这是 MDG 的终极验证。

### 三、诚实纪律（硬约束）
- 著名案例/历史盲测**不进真钱校准桶**（上岗考基线与真钱分开记）；
- 同一标的、同一投资论点拆出的多个微预测，**按论点聚类只计一次**（防伪造样本量）；
- Brier 报告须并报，n 未达阈值时只呈现不判读。

### 四、跨模型对接（本制度的核心价值）
每单交卷后，把四件摘要随报告回传（经委托人摆渡）：**MDG+降级信号 + 冻结概率分布 + 双向证伪信号 + 结算日**；结算时再回传**实际归类结果 + 护城河是否失守**。用途：对侧据此建"跨模型对照台账"，季度出三张表——
1. 你的校准桶（独立计，对侧不碰你的参数）；
2. 对侧校准桶；
3. **跨模型一致性表**：①两侧独立**同判**的案子事后一起对/一起错——**一起错的案子是两个模型的共同盲点，最危险，单独列出；其中"两侧同判高MDG而事后失守"是最高危共同盲点，最优先列**；②两侧**异判**的案子事后谁对——分歧案的胜率，衡量谁的独立判断更值钱。
**注意：两侧同判不代表更可信**（可能一起错）；真正的高价值信号是分歧，以及分歧最后谁对。所以**不要因为知道对侧也这么判就调高自己的置信**——你交卷前看不到对侧，交卷后也不据此改分。
