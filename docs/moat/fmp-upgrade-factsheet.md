## 三、是否真治本——对照 ADJ-0718-01① 指数成分展开法（CGM 事实对照·非推销）
- **① 指数成分展开法**（S&P 500＋MidCap 400＋SmallCap 600 成分展开≈1500名·`indexes` 端点**现档即可用**·逐名核数走 **SEC EDGAR XBRL**〔本会话已证可靠·公开免费〕）＝**零成本·结构性治本**：全域底册由指数成分给定、基本面由 SEC XBRL 给定，**两者都不依赖 FMP 付费**。
- **FMP 升级之边际价值**＝主要省"手工枚举全域名单"之便利，以及 market-cap/statements 的一站式取数便利；但**底册可由 indexes 免费得、核数已由 SEC XBRL 免费得**——故升级**非治本必需**，属"提速/便利"投资，非"解锁不可得数据"。
- **例外情形**（升级边际价值升高之条件）：若日后需要 FMP 独有且 SEC XBRL 无法替代之数据（如实时/盘中、非美申报人结构化基本面、大规模 bulk 拉取以缩滚扫周期），则升级价值上升——**欧陆四名（非 SEC 申报人）之结构化基本面正是 SEC XBRL 覆盖不到的缺口**，但该缺口 FMP 亦未必补齐（本会话 FMP 对欧陆亦仅 profile 级），故欧陆走年报直读（ADJ-0718-01③）而非靠 FMP 升级。
