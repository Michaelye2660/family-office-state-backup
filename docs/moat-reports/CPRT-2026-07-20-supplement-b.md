## 二、土地/设施租赁专节（本补证核心问题）

**问题**：Copart 现在是否向创始家族（或其实体）租赁任何堆场/设施？

**答案（以 2025-10-24 DEF 14A 为准）：无。零条。**

核验过程：
1. 2025 DEF 14A 全文（3,969 行纯文本）检索 "lease / landlord / rent"——**所有命中均为公司飞机（corporate-owned or leased aircraft）高管个人使用计价方法的描述**，无任何不动产租赁关联交易条目。
2. Related Person Transactions 节仅列上表 2 条亲属雇佣，无租赁、无购售、无借贷、无回购类条目。
3. 依 SEC Reg S-K Item 404(a)，与董事/高管/5% 股东（及直系亲属）之间金额 >$120,000 的交易**强制披露**；Willis Johnson 仍持股 5.75% 在披露范围内；任何堆场级年租金必然远超 $120k 门槛。**推论（标注为推论）**：当前 281 处全球设施（FY25 10-K Item 2 口径）中向创始家族租赁的数量 = **0 处，占比 0%**——此为"披露缺失即不存在"的法定推论，非公司明文否认句。
4. "年租金/租期/独立评估"诸问，因**现无此类租赁**而不适用；历史上存在时的对应数据见第三节。

### 3.4 演变结论（事实定性，非投资判读）

- **性质=历史遗留，且已清（cleared），非持续存在。**
- 时间线：1992 起 Johnson（Fresno）/Meeks（San Martin）/Schmidt（LA）三处家族/董事关联租赁 → Schmidt 于 ~2006 前退出披露、Meeks 租约 2007-08 到期 → 最后一处（创始人 Johnson 的 Fresno）于 **2010-11-15 由公司按独立评估价 $1.8M 买断** → **2011–2025 连续 15 年 proxy 中土地/设施类关联租赁为零条目**（本席实核 2011、2016、2018、2023、2024、2025 六份；2012–2015、2017、2019–2022 各年未逐份核验，标注于第五节）。
- 与 T2 "不可复制土地"标签的交叉核验结果：**现行 281 设施口径下，护城河土地不含向创始家族租赁的关联方资产**（法定披露推论，见第二节第 3 点）。历史上规模峰值亦仅 3 处、年租金合计 <$45 万量级（FY03：$154,008+$60,923+$228,000 ≈ $442,931），且末笔以独立评估价买断。

## 五、未能核验清单

1. **2012–2015、2017、2019–2022 各年度 proxy 的 RPT 节未逐份照录**——已核 2003/2006/2010/2011/2016/2018/2023/2024/2025 共九份，覆盖起点、清仓点与近三年；中间未核年份存在"曾短暂出现其他关联交易"的残余可能（但土地租赁在 2011 年买断后复现的概率受连续六份实核样本约束）。如陪审要求"逐年无缝"，可补。
2. **"占 281 设施比重=0" 属法定披露推论而非公司明文陈述**——proxy 不会写"我们不向家族租地"；0 的结论依赖 Item 404(a) 强制披露完整性 + Johnson 仍在 5% 披露圈内这两个前提。FY25 10-K 财报附注中的 related party 脚注本席未另行调取（proxy 为 Item 404 的法定披露载体，10-K 脚注为交叉冗余源）。
3. **"Jeff Pocock" 无此人事任命**——2026 年 6–7 月两份 8-K 中总裁任命对象为 **Jane** Pocock；"Jeff Pocock" 表述未能核验（判定为人名讹误，非另有其人另有任命）。
4. Brett Adair / Diane Yassa 雇佣条款"与同岗位员工可比"仅有公司自陈（"terms … no less favorable"），无第三方薪酬对标可核——此为所有 proxy RPT 披露的固有边界，非本案特有缺口。

## 六、核录小结（纯事实，供案卷 v2 引用；零投资结论）

1. 2025-10-24 DEF 14A 的 Related Person Transactions 节**全文仅 2 条，均为亲属雇佣**（Brett Adair $410,905；Diane Yassa $326,993），全部经全独立董事构成的审计委员会**事先**审批；**无任何土地/设施租赁条目**。
2. 陪审所指"创始家族关联方土地租赁"确为真实历史（1992–2010，峰值 3 处、年租金合计约 $44.3 万），但**最后一处（创始人 Johnson 的 Fresno 设施）已于 2010-11-15 按持牌评估师评估价 $1.8M 被公司买断**，此后 15 年（实核六份 proxy）零条目——**T2"不可复制土地"与家族关联租赁在当前 281 设施口径下不存在交叉**（法定披露推论）。
3. "Jeff Pocock 升总裁"经 8-K（0001193125-26-298722）核为 **Jane Pocock，2026-08-01 生效**属实、人名有误；同期 8-K（0001193125-26-286982）载 Jay Adair 2026-07-31 回任 CEO、Jeffrey Liaw 卸任——治理判读上移。
